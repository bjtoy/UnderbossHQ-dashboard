THE GRAND MAFIA
Faction Edition — Raid Preparation & Execution
:::

:::text-lead
Raids are coordinated faction attacks against high-value PvE or PvP targets — enemy turfs, Kingpins, Mercenaries, and event bosses. Multiple members feed one rally. One player leads it.
:::

:::color-gold
Successful raids are won before the rally launches.
:::

:::warning
The rally leader's gear, enforcers, combat specs, and boosts determine the raid's overall strength. Joiners amplify — they do not rewrite — the leader's setup.
:::

:::section-bar
WHAT RAIDS ARE & WHY THEY MATTER
:::

:::text-body
Raids win resources, denial, and faction growth when the whole crew shows up prepared:
:::

• Faction dominance in turf wars and events
• Resource denial and enemy suppression
• High-tier loot from PvE raid events
• Real coordination practice under pressure

:::section-bar
RAID PREPARATION — RALLY LEADERS
:::

:::heading-md
Checklist before you open the rally
:::

• **Mansion Level:** Higher levels = larger rally capacity
• **Crew Capacity:** Maximize via investments and Godfather gear
• **Gear:** Equip a full 6-piece combat set (Victor, New Blood, Kings, Godfather)
• **Godfather Equipment:** Piece that boosts your main troop type or rally size
• **Combat Specs:** Combat Spec preset — not Economy or Kingpin
• **Boosts:** 25% or 50% ATK, DEF, HP, and March Size
• **Enforcers:** Four that match your troop type and synergize
• **Troop Type:** Prefer mono-type (e.g. all Bikers) for synergy
• **Scout:** Reinforcements, troop types, traps, anti-scout

:::important
Lock the rally 10–15 seconds before launch. Screenshot your setup for post-raid review.
:::

:::section-bar
RAID PREPARATION — RALLY JOINERS
:::

:::heading-md
How to feed the rally without sabotaging it
:::

• **Troop Type:** Send only the type the leader called
• **No Enforcers:** Only the leader's enforcers apply
• **Combat Specs:** Combat Spec for the troop type you send
• **Boosts:** Optional for PvE; recommended for PvP
• **Troop Tier:** T9 or T10 for maximum impact
• **Join Early:** Late joins may be cut from the march

:::warning
Only the rally leader's enforcers are used. Wrong troop type = wasted march and weaker raid.
:::

:::section-bar
GEAR SETS
:::

:::heading-md
Combat sets
:::

**VICTOR** — Biker ATK/DEF, March Speed — UB 50+ — Biker raids

**NEW BLOOD** — Shooter ATK/HP — UB 50+ — Shooter raids

**KINGS** — Rally Capacity, all troop stats — UB 60+ — High-end raids

**GODFATHER** — Balanced ATK/DEF/HP — UB 60+ — General PvP

:::heading-md
PvE / early
:::

**CHALLENGER** — Kingpin damage, Enforcer ATK — Early game — Kingpin fights

:::section-divider
SET BONUSES
:::

• **2 pieces:** Minor boost (e.g. +5% ATK)
• **4 pieces:** Moderate boost (e.g. +10% DEF)
• **6 pieces:** Major boost (e.g. +15% HP, March Size)

:::tip
Only matching set pieces count. Mixing sets kills the set bonus — go full six of one combat set for raids.
:::

:::section-bar
COMBAT SPECS FOR RAIDING
:::

:::heading-md
Rally leaders — prioritize
:::

• Troop ATK / DEF / HP for your main troop type
• Rally Capacity
• Troop Damage Bonus
• Enemy Damage Reduction

:::heading-md
Rally joiners — focus on
:::

• Troop ATK / DEF / HP for the type you're sending
• March Speed (optional)
• Ignore Rally Capacity — that only matters for leaders

:::important
Double-check your active spec before launch or join. Use presets to flip between Combat, Kingpin, and Economy.
:::

:::section-bar
BOOSTS TO ACTIVATE
:::

• **Attack (25% / 50%):** Leaders — mandatory. Joiners — PvP recommended
• **Defense:** Reduces incoming damage — optional, mainly PvP
• **Crew HP:** Survivability — useful in PvP
• **March Capacity:** More troops you can send — leaders only
• **Rally Capacity:** Bigger rally — leaders via gear / specs

:::section-pill
TROOP COMPOSITION & VARIANTS
:::

:::heading-md
Mono-type
:::

:::text-body
All one troop type (e.g. Bikers). Default for most raids — best synergy and peak damage.
:::

:::heading-md
Mixed-type
:::

:::text-body
Two types (e.g. Biker + Bruiser). Use when you need a specific counter to the target's setup.
:::

:::heading-md
Tank-heavy
:::

:::text-body
Bruiser-weighted frontline. Breaking traps or holding a defense.
:::

:::heading-md
Speed raid
:::

:::text-body
Biker-focused, fast march — tile hits, bait, and snappy PvP.
:::

:::section-bar
RAID EXECUTION
:::

:::heading-md
Scouting
:::

:::text-body
Scout before you commit. Look for reinforcements, anti-scout, and troop mix. Cancel or retarget if it's a trap.
:::

:::heading-md
Communication
:::

• Assign roles: rally leader, reinforcers, backup rallies
• Share rally links early
• Confirm troop type and joiner readiness in faction chat / Discord

:::heading-md
Timing
:::

• Lock a launch window everyone can hit
• Teleport near the target before open
• Fake rallies to bait or split enemy response when it matters

POST-RAID STRATEGY
:::

• Review battle reports — performance, damage, losses
• Heal wounded troops immediately
• Share reports with the faction
• Adjust enforcers, specs, or gear from what you learned
• Queue follow-up raids or reinforcements while the window is open
