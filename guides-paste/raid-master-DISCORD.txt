# THE GRAND MAFIA
**Faction Edition — Raid Preparation & Execution**

Raids are coordinated faction attacks against high-value PvE or PvP targets (enemy turfs, Kingpins, Mercenaries, event bosses). Multiple members feed one rally. One player leads it.

🏆 **Successful raids are won before the rally launches.**

⚠️ **The rally leader's gear, enforcers, combat specs, and boosts determine the raid's overall strength.** Joiners amplify — they do not rewrite — the leader's setup.

## What raids are & why they matter
• Faction dominance in turf wars and events
• Resource denial and enemy suppression
• High-tier loot from PvE raid events
• Real coordination practice under pressure

## Raid preparation — Rally leaders
• **Mansion Level:** Higher = larger rally capacity
• **Crew Capacity:** Maximize via investments + Godfather gear
• **Gear:** Full 6-piece combat set (Victor / New Blood / Kings / Godfather)
• **Godfather Equipment:** Boost main troop type or rally size
• **Combat Specs:** Combat preset (not Economy / Kingpin)
• **Boosts:** 25% or 50% ATK, DEF, HP, March Size
• **Enforcers:** 4 matching troop type + synergy
• **Troop Type:** Mono-type preferred (e.g. all Bikers)
• **Scout:** Reinforcements, troop types, traps, anti-scout

❗ Lock the rally 10–15s before launch. Screenshot your setup.


—— PART 2 / 4 ——
## Raid preparation — Rally joiners
• Send **only** the troop type the leader called
• **No enforcers** — leader's enforcers only
• Activate **Combat Specs** for the type you send
• Boosts: optional PvE / recommended PvP
• Troop tier: **T9 or T10**
• Join early — late joins may be cut

⚠️ Wrong troop type = wasted march and a weaker raid.

## Enforcer strategy (leaders)

**Biker** — Vlad · Goro · Angelo · Yuki — burst, speed, crit
**Shooter** — Scarlet · Tracy · Eva · Yuki — long-range AoE + P.ATK
**Bruiser** — Donald · Klaas · Bobby · Gina — HP, sustain, frontline

> Yuki is the flex pick when a fourth seat is open.

## Gear sets
• **Victor** — Biker ATK/DEF, March Speed — UB 50+ — Biker
• **New Blood** — Shooter ATK/HP — UB 50+ — Shooter
• **Kings** — Rally Capacity + all troops — UB 60+ — High-end
• **Godfather** — Balanced ATK/DEF/HP — UB 60+ — General PvP
• **Challenger** — Kingpin dmg / Enforcer ATK — Early — Kingpins

## Set bonuses
• 2pc minor · 4pc moderate · 6pc major (e.g. HP / March Size)
> Only matching pieces count. Mixing sets kills the bonus.

## Combat specs
**Leaders:** troop ATK/DEF/HP (main type), Rally Capacity, Troop Damage Bonus, Enemy Damage Reduction
**Joiners:** troop ATK/DEF/HP (sent type), March Speed optional — Rally Capacity is leaders-only

❗ Double-check active spec before launch/join (Combat / Kingpin / Economy presets).

## Boosts
• **Attack 25%/50%** — leaders mandatory; joiners for PvP
• **Defense** — optional PvP
• **Crew HP** — useful PvP
• **March Capacity** — leaders only
• **Rally Capacity** — leaders (gear/specs)

## Troop composition variants
• **Mono-type** — one type — default / max synergy
• **Mixed-type** — two types — specific counters
• **Tank-heavy** — Bruiser frontline — traps / defense
• **Speed** — Biker-focused — tiles, bait, fast PvP

## Raid execution
**Scout first:** reinforcements, anti-scout, troop mix. Cancel/retarget if trap.

**Communicate:**
• Roles: leader / reinforcers / backup rallies
• Share rally links early
• Confirm troop type + joiner readiness

**Timing:**
• Lock a launch window
• Teleport near target
• Fake rallies to bait / split response when needed

## Final pre-raid checklist (leader)
☐ Full 6-piece combat set
☐ Godfather gear for troop ATK or Rally Size
☐ Combat Specs (not Economy / Kingpin)
☐ Boosts: ATK, DEF, HP, March Size
☐ 4 synergistic enforcers matching troop type
☐ Troop type called to joiners
☐ Final scout
☐ Teleport near target
☐ Lock rally 10–15s before launch
☐ Screenshot setup

## Post-raid
• Review reports (performance / damage / losses)
• Heal wounded immediately
• Share with faction
• Adjust enforcers / specs / gear
• Queue follow-ups while the window is open

🏆 **Prepare correctly. Coordinate clearly. Analyze results. Improve continuously.**
